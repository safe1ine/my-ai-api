# AI API 中转站 · 需求文档

## 一、产品概述

AI API 中转站是一个 API 代理服务，支持将请求转发到 OpenAI 或 Anthropic 上游，并提供统一的管理后台。用户使用本站颁发的 API Key 进行调用，系统自动路由到对应上游并记录每次调用日志。

---

## 二、功能模块

### 2.1 统计

**概述指标**
展示全局汇总数据，包括：总请求数、Input Tokens、Output Tokens、Total Tokens。

**Token 趋势**
折线图展示一段时间内的 token 用量变化，分 Input 和 Output 两条线。支持按日、周、月三个粒度切换。

**模型分布**
环形饼图展示各模型的 token 占比，图上标注模型名和百分比。

**API Key 排行**
表格展示各 API Key 的请求数和总 token 用量。

---

### 2.2 调用记录

展示所有历史调用记录，每条记录包含：调用时间、成功/失败状态、模型名、所用上游、API Key（脱敏）、token 用量（Input + Output = Total）、耗时。

支持按模型名关键词搜索，以及按成功/失败状态筛选。分页展示，每页 20 条。

---

### 2.3 上游管理

管理转发目标的上游服务商配置。

**列表**：展示所有已配置的上游，包含名称、类型、API Key 前缀、Base URL、启用状态、创建时间。

**新增/编辑**：配置项包括名称、类型（OpenAI 或 Anthropic）、API Key、Base URL（可选）、是否启用。编辑时自动回填当前 API Key。

**测试连通性**：对指定上游发起连通测试，以提示告知成功或失败。

**查看模型列表**：拉取并展示该上游支持的模型名称列表。

**删除**：删除前需二次确认。

---

### 2.4 API Key 管理

管理调用方使用的 API Key。

**列表**：展示所有 Key 的名称、脱敏后的 Key 值、启用状态、创建时间。

**新建**：输入名称后生成 Key。Key 仅在创建成功后完整展示一次，用户须自行复制保存，之后只显示脱敏版本。

**启用/禁用**：可随时切换每个 Key 的启用状态。

**删除**：删除前需二次确认。

---

### 2.5 使用说明

提供接入指引，包括：
- 本站的接入地址（Base URL）
- 请求鉴权方式
- OpenAI 兼容接口的调用示例（含模型路由说明：模型名以 `claude` 开头时自动转发至 Anthropic）
- Anthropic 原生接口的调用示例
- 快速开始步骤

所有示例代码均可一键复制。
